# Composite Services Allowlist

Serviços compostos aprovados (interfaces `I*Service` que agregam `*CommandService` + `*QueryService`).
Novas entradas exigem atualização de ADR e revisão arquitetural.

- `ICollaboratorService`
- `IMetricCheckinService`
- `IMissionMetricService`
- `IMissionService`
- `ITeamService`
- `IWorkspaceService`
